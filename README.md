# Dispatch

Open source OTA updates.

## About

A web-based platform for managing OTA updates focused on simplicity, ease of use, and self-hosting.

Uses Hawkbit DDI API as a wire format so is compatible with RAUC and SWUpdate.

Very much in Alpha.
